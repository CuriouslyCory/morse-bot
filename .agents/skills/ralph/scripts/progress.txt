## Codebase Patterns
- Use `npx expo install <packages>` to resolve SDK-compatible versions (auto-resolves based on expo version in package.json)
- sherif enforces alphabetical ordering in package.json deps; always sort alphabetically
- Use `catalog:` for packages in pnpm-workspace.yaml catalog, explicit versions for others
- `@morse-bot/morse-decoder` is a workspace package: use `workspace:*` as version
- For Expo SDK 54, expo-clipboard is ~8.x, expo-document-picker is ~14.x, expo-file-system is ~19.x, expo-sharing is ~14.x
- react-native-live-audio-stream@^1.1.1 and react-native-audio-api@^0.11.6 are the versions compatible with this project
- @shopify/react-native-skia@2.2.12 and @react-native-community/slider@5.0.1 are SDK 54 compatible
- Never use .js extensions in TS imports (bundler module resolution)
