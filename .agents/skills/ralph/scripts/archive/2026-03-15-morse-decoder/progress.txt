## Codebase Patterns
- This is a pnpm monorepo using Turborepo. Apps in apps/, shared packages in packages/, tooling in tooling/.
- sherif@latest enforces alphabetical ordering in package.json dependencies/devDependencies. Always sort keys alphabetically.
- pnpm postinstall runs sherif lint:ws automatically; fix ordering issues before committing.
- Package scope is @morse-bot/ (was @acme/ in the t3-turbo template).
- Root package.json name is "morse-bot".
- The `public/` directory in apps/nextjs must be excluded from tsconfig.json to avoid TS errors on plain JS worklet files.
